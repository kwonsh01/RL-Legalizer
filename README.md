# RL-Legalizer
*OpenDP with PPO by SH Kwon, SH Kim, CH Lee*

### Required
* Python, SWIG

### How To Compile
    $ git clone --recursive https://github.com/kwonsh01/RL-Legalizer.git
    $ cd RL-Legalizer
    $ mkdir build
    $ cd build
    $ cmake ..
    $ make

### How To Execute
    $ cd rldp
    $ python3 PPO.py
    $ cd data
    $ python3 data.py
